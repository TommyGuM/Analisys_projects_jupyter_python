(Big Goal)

Оптимизация LTV и поиск точек роста выручки для [Название сферы]

 Компания столкнулась с падением среднего чека у самых активных пользователей. 
 
 Цель проекта — сегментировать базу и предложить стратегию удержания (Retention).
 
 Стек технологий  Python (Pandas, Scipy, Plotly), SQL, А/Б тестирование.
 
 Ключевые этапы :
ETL & Cleaning: Объединение разрозненных логов и транзакций в единый профиль клиента.
EDA: Выявление аномалии — «Power Users» приносят меньше прибыли, чем новички.
Hypothesis: Предположение, что текущая модель подписки не выгодна для тяжелого контента.
A/B Test Simulation: Расчет необходимого размера выборки и стат-значимости изменений.

 Главный инсайт
 «Анализ показал, что внедрение реферальной программы для сегмента 'Power Users' может снизить стоимость привлечения (CAC) на 15%».
